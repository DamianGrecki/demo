# Demo Spring Boot + PostgreSQL

---

## Wymagania

- Java 25+
- Maven 3+
- Docker
- IDE (np. IntelliJ)

---

## 1. Uruchomienie bazy danych w kontenerze

W katalogu projektu uruchom:

```bash
docker-compose up -d
```
## 2. Uruchomienie aplikacji

W katalogu projektu uruchom:

```bash
./mvnw spring-boot:run
```
